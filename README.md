# Lua Deobfuscator Framework

Questo repository contiene uno script Lua per deoffuscare altri script Lua.

## Come usare

1. Inserisci lo script offuscato in `input.lua`.
2. Esegui `lua src/main.lua`.
3. Troverai il risultato in `dump/output.lua`.

## Funzionalità

- Rileva offuscamento base64
- Pulisce spazi e linee vuote
- Facile estendibilità per altri tipi di offuscamento
